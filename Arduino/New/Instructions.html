Instructions for starting ASCEND! Ground Station

General Startup

Unbox station and start computer

Plug in GS(Ground Station) and make sure the power switch is in the off position

Plug the GS into the wall

Connect the computer to the GS through the other port on the GS

Grab the featherweight GS and featherweight GPS Tracker and attach the antennas to them

Computer Startup

Startup Arduino IDE and open Arduino_Ground_Station.ino

INSERT VS CODE STARTUP HERE

When Arduino is open upload the code to the ground station and make sure everything has turned on in the Arduino GS

[IMPORTANT] TURN OFF SERIAL MONITOR IN Arduino IDE

Once computer is connected to GS run Serial_Read_Send_WINDOWS.c

Open Device Manager and check COM ports

Find which COM ports connected to GS featherweight and GS Arduino

In where you ran Serial_Read_Send_WINDOWS.c There should be a input to put in input and output COM ports put GS featherweight COM port in input and GS Arduino COM port in output

featherweight Startup

Turn on featherweight GS

Turn on featherweight GPS Tracker [AFTER THE GROUND STATION IS ON]

If not already plug in the featherweight GS into the computer

Ground Station Startup

Turn the switch near the power cable to on [AFTER ARDUINO HAS BEEN PLUGGED IN AND UPLOADED TO]
